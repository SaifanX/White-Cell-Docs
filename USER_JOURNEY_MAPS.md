## Study Module Journey

### Phase 1: Onboarding (Days 1-3)

**Goal:** Learn the study system and create first deck

```
User arrives → Welcome video → Create account → Verify email
    ↓
Guided tour of Study module → Create first deck
    ↓
Add 10 sample flashcards → Review prompt → Learn SM-2 algorithm
    ↓
Complete first review session → Earn 100 XP
```

**Key Metrics:**
- Flashcards created: 10
- First review: Yes
- Tutorials completed: 2/2
- XP earned: 100

**Decision Points:**
- Do they create own deck or use template?
- Do they prefer spaced repetition explanation?

---

### Phase 2: Building Knowledge (Weeks 1-2)

**Goal:** Establish daily study habit

```
Daily login → Check due cards → Study session
    ↓
Add new notes → Create flashcards from notes
    ↓
Take quiz on topic → Review weak areas
    ↓
Earn streak badge → Share progress
```

**Milestones:**
- **Day 7:** 7-day streak badge
- **Card count:** 50+ cards
- **Review count:** 100+ reviews
- **XP:** 500+ XP earned

**Pain Points & Solutions:**
- Pain: Forgot to review cards
  - Solution: Daily reminders via push notification
- Pain: Creating flashcards takes too long
  - Solution: AI quiz generation tool
- Pain: Can't find notes later
  - Solution: Tagging and search system

---

### Phase 3: Mastery Building (Weeks 3-8)

**Goal:** Build mastery across multiple subjects

```
Multiple decks in progress → Balanced review schedule
    ↓
Deep learning with notes + quizzes → Progress tracking
    ↓
Switch to study sessions → Export notes as study guides
    ↓
Achieve subject mastery → Celebrate with badge
```

**Milestones:**
- **Card count:** 200+ cards
- **Decks created:** 5+
- **Quiz score:** 80%+ average
- **XP:** 2,000+ XP earned
- **Achievement:** "Subject Master" badge

**Engagement Activities:**
- Weekly review stats
- Comparison with study goals
- Leaderboard position
- Content sharing

---

### Phase 4: Optimization (Months 2+)

**Goal:** Maintain knowledge and optimize learning

```
Analyze study patterns → Identify weak topics
    ↓
Increase difficulty → Use advanced features
    ↓
Share learning resources → Help other learners
    ↓
Archive mastered decks → Focus on new topics
```

**Advanced Features Used:**
- Custom study schedules
- Difficulty levels
- Note exports
- Quiz generation

**Retention Metrics:**
- Monthly active users in Study: 85%+
- Average session duration: 15-20 minutes
- Cards reviewed per day: 20-30

## Fitness Module Journey

### Phase 1: Getting Started (Week 1)

**Goal:** Log first workout and understand fitness tracking

```
Browse Fitness module → View sample workouts
    ↓
Set fitness goal (lose weight/build muscle/general health)
    ↓
Log first workout → Add exercises with weight/reps
    ↓
View workout summary → Earn 50 XP
```

**Initial Setup:**
- Goal selection: 3 options
- Unit preference: kg or lbs
- Activity level: sedentary to very active
- Fitness baseline: no experience → advanced

**Decision Points:**
- Guided workout vs. free logging?
- Track nutrition?
- Use timer for cardio?

---

### Phase 2: Building Consistency (Weeks 2-4)

**Goal:** Establish 3x per week workout habit

```
Daily activity check-in → Log today's status
    ↓
Complete workouts → Track exercises and weights
    ↓
Monitor progress photos → Measure body metrics
    ↓
Earn streak badges → Share achievements
```

**Milestones:**
- **Week 1:** First 5 workouts
- **Week 2:** 10 workouts, 1 measurement
- **Week 4:** 12+ workouts, streak badge
- **XP Earned:** 500+

**Engagement Triggers:**
- "You're 2 days away from a 7-day streak!"
- Strength increase notifications
- Weekly summary email

---

### Phase 3: Peak Performance (Months 2-3)

**Goal:** Achieve personal records and build habit

```
Consistent 4-5 day per week routine → Track PRs
    ↓
Add nutrition tracking → Log meals and calories
    ↓
Analyze performance trends → Optimize training
    ↓
Achieve PR badge → Update fitness goals
```

**Metrics:**
- **Workouts:** 16-20 per month
- **PRs achieved:** 3+
- **Nutrition days logged:** 50+
- **Progress photos:** 4+ (monthly updates)
- **XP:** 2,000+

**Advanced Features:**
- Workout templates
- Nutrition analysis
- Performance charts
- Goal achievement tracking

---

### Phase 4: Long-term Success (Month 4+)

**Goal:** Maintain fitness and continuous improvement

```
Master plateau → Adjust goals → New training plan

Long-term progress visualization → Celebrate milestones
    ↓
Help others → Share training tips

```

**Retention Metrics:**

## Cross-Module Mastery Path

### Level Progression System

```
Level 1: Beginner (0-500 XP)
├─ Complete 1 task in each module
├─ Create 5 flashcards
├─ Log 1 workout
└─ Total XP earned: 500

Level 2: Apprentice (500-1,500 XP)
├─ 7-day streak in Discipline
├─ 50 flashcards mastered
├─ 10 workouts logged
├─ 1 content piece published
└─ Total XP earned: 1,500

Level 3: Expert (1,500-5,000 XP)
├─ 30-day streak in Discipline
├─ 200 cards mastered
├─ 40 workouts logged
├─ 10 content pieces published
└─ Total XP earned: 5,000

Level 4: Master (5,000+ XP)
├─ Multiple 60+ day streaks
├─ 500+ cards mastered
├─ 100+ workouts logged
├─ Creator with 500+ followers
└─ Total XP earned: 10,000+
```

### Skill Badges

**Total Badges Available: 40+**

- **Study Badges:** 8
  - Flashcard Master, Quiz Ace, Note Taker, etc.
  
- **Fitness Badges:** 8
  - PR Breaker, Consistency King, Endurance Champion, etc.
  
- **Mind Badges:** 8
  - Chess Master, Speedcubing Expert, Debater, etc.
  
- **Discipline Badges:** 8
  - Streak Builder, Habit Master, Resilience, etc.
  
- **Creator Badges:** 8

## User Segments

**Characteristics:**
- Age: 18-25
- Goal: Better grades and exam prep
- Time availability: 30-60 min/day
- Primary modules: Study (90%), Mind (60%), Discipline (70%)

**Journey Focus:**
- Fast flashcard creation
- Quiz preparation
- Exam scheduling
- Grade tracking

**Retention Strategy:**
- Achievement badges for each exam
- Study group features
- AI homework help

- Goal: Build/maintain fitness
