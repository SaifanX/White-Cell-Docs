## Shortcut Hints

Shortcuts are displayed in several places:

1. **Tooltips**: Hover over buttons to see shortcut
2. **Command Palette**: ⌘K shows all available commands
3. **Help Menu**: ? shows context-sensitive help
4. **Settings**: View all shortcuts in one place
5. **Onboarding**: New users shown key shortcuts

---

## Accessibility Notes

- All shortcuts have mouse/touch alternatives
- No single-key shortcuts (except numbers 1-5 in specific contexts)
- Modifiers clearly indicated (⌘Cmd, Ctrl, Shift, Alt)
- Shortcuts don't interfere with browser defaults
- Screen reader support included for all actions

## Implementation Notes

Shortcuts are implemented using:
- `cmdk` library for command palette
- `useKeyPress` hook for keyboard events
- `useHotkeys` for global shortcuts
- Accessibility-first approach
